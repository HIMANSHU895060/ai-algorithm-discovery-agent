===================================================================

===================================================================
TROUBLESHOOTING
===================================================================

PROBLEM: INSTALL.bat doesn't work
SOLUTION: Download and install Python first
  - Go to: python.org
  - Click "Downloads"
  - Download "Python 3.9" or higher
  - Run the installer
  - CHECK the box: "Add Python to PATH"
  - Then run INSTALL.bat again

PROBLEM: Command window closes too fast
SOLUTION: Use RUN.bat instead of typing commands
  - RUN.bat keeps the window open
  - It shows "pause" at the end
  - Press any key to close

PROBLEM: "File not found" errors
SOLUTION: Make sure you're in the right folder
  - Open the folder with all the files
  - Double-click the .bat files from INSIDE that folder

===================================================================
WHAT DOES IT DO?
===================================================================

1. DISCOVER: Finds the best algorithm for your problem
   Input: Problem type (sorting, searching, etc.)
   Output: Best algorithm, complexity, fitness score

2. OPTIMIZE: Makes algorithms faster using genetic algorithms
   Uses evolution to improve parameters

3. LEARN: Uses AI (Q-Learning) to learn which algorithms work best
   Gets smarter every time you use it

4. SAVE LOCALLY: Keeps everything on your computer
   No cloud, no internet needed
